# 🏨 Hotel Management System

A full-stack web application built with **Java Spring Boot, Angular 18, MySQL, and Ng-Zorro** for managing hotel operations, including room booking, customer management, and authentication.

---

## 🌍 Live Demo  
🚀 **[Project Repository](https://github.com/sarthakpawar0912/Hotel-Management-Web.git)**  

---

## 🌟 Features  

✔️ **User Authentication** – Secure login and registration using **Spring Security & JWT**.  
✔️ **Room Management** – View, add, edit, and manage room inventory.  
✔️ **Booking Management** – Book rooms for specific dates and track reservations.  
✔️ **Customer Management** – Add, update, and manage customer details.  
✔️ **Admin Dashboard** – A central interface for hotel management tasks.  
✔️ **Responsive UI** – Built with **Angular 18 & Ng-Zorro** for a modern experience.  
✔️ **RESTful API** – Backend services powered by Spring Boot with Hibernate for database interactions.  
✔️ **Role-Based Access Control** – Admin and customer roles with specific privileges.
